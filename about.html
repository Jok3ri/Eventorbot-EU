The aim of this project is to recreate the documentation of the Eventorbot! Open source 3D printer in OpenSCAD format. Although the printer itself is a free project, it has been documented with Google's SketchUp program which isn't free at all and can't work under GNU/Linux (or any other free operating system probably), even with the help of Wine.
